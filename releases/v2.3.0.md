# Pocket SCION Synth v2.3.0

First public release of the alternative real-time synthesis firmware.

## Included

- 128 patches in eight banks, plus a separately selectable pitch-bend bank
- Four-voice PRA32-U synthesis with live envelopes, filters, LFOs, chorus,
  and delay
- Three sensor-driven Euclidean rhythm lanes with variable ratcheting
- DIN and USB MIDI output, including single- and multichannel modes
- Pressure-to-pitch Raw Output Mode
- Nine-pixel RGB animation driven by live envelopes, LFOs, polyphony, and
  ratchets

## Installation

Download `pocket_scion_synth_v2.3.0.uf2`, enter the RP2040 USB bootloader,
and copy the file to the `RPI-RP2` volume.
