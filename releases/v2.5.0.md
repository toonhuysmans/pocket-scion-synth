# Pocket SCION Synth v2.5.0 — Adaptive Calibration

This release expands the alternative multi-timbral firmware to sixteen banks
and makes the plant/touch sensor pipeline observable and tunable from the USB
editor.

## New in v2.5.0

- 256 deliberately designed patches across sixteen banks, including Dub
  Techno, Motorik, Polyrhythmic Organic, Cinematic, Acid & Electro, Broken Beat
  & IDM, Minimal Phase, and Chiptune directions
- Persistent controls for response-window length, edge-noise rejection,
  adaptive normalization, pressure/expression smoothing, variation gain,
  transient response, activity timeout, calibration learning, and calibration
  recovery
- Faster 1.0%-per-window default recovery after temporary sensor spikes, with
  an adjustable 0.1–5.0% range
- Live 10/30/60-second pressure, expression, transient, and pitch-motion graph
- Calibration plots showing current pulse interval and relative variation
  against their learned minimum–maximum ranges
- Live Learn/Freeze, Active/Idle, analysis-rate, rejected-edge, dropped-edge,
  and signal-age diagnostics
- Calibration controls integrated into the Sensor Monitor for immediate visual
  feedback
- Backward-compatible migration of existing saved global settings

## Installation

Download `pocket_scion_synth_v2.5.0.uf2`, enter the RP2040 USB bootloader, and
copy the file to the `RPI-RP2` volume. Existing v2.4.0 patch and global records
remain readable; newly added calibration values begin at their documented
defaults until saved.

SHA-256:

```text
3397c4337ca47e358c64ad296834d90c62933368de25bd8d5ba52950d56023f6
```

The previous [v2.4.0 multi-timbral release](https://github.com/toonhuysmans/pocket-scion-synth/releases/tag/multi-tibral)
remains available for rollback. The official Instruō factory firmware can be
restored using the links in the project README.
