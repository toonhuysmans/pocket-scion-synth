# Pocket SCION Synth v2.4.0 — Multi-Timbral

This release turns the three generative lanes into distinct instruments while
keeping the live, sensor-driven character of the alternative firmware.

## Included

- Independent PRA32-U timbres for bass/percussion, pad, and lead
- Fully parameterized 128-patch bank across eight deliberately designed banks
- Configurable Bass, Percussion, or Hybrid low role with six editable
  articulation slots
- Separate Euclidean density, length, timing, motif, envelope, and sensor
  routing controls for all three lanes
- Shared chorus and stereo delay on the complete three-part mix
- DIN and USB MIDI, single- and multichannel modes, ratchets, and held-chord
  scale constraint
- Browser editor with live control, bank/patch JSON, flash save, parameter help,
  signal-flow documentation, and automatic patch synchronization
- Raw sensor audio mode, responsive nine-pixel RGB animation, and a 20 Hz live
  sensor monitor

## Installation

Download `pocket_scion_synth_v2.4.0.uf2`, enter the RP2040 USB bootloader,
and copy the file to the `RPI-RP2` volume.

SHA-256:

```text
ce367a389a3248ab620f98aaede7c0e3b3c2ef7855b558953ef10dcdeb0ae29f
```

The previous [v2.3.0 single-timbre release](https://github.com/toonhuysmans/pocket-scion-synth/releases/tag/v2.3.0)
remains available for comparison or rollback. The official Instruō factory
firmware can be restored using the links in the project README.
